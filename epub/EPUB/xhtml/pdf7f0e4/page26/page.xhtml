Características del método científico

El conocimiento científico se genera a partir un proceso que tiene como

finalidad el establecimiento de relaciones entre diferentes hechos el cual

es denominado método científico. Se caracteriza por ser falseable (los

principios científicos siempre estarán sometidas a ser reevaluados como

falsos) y reproducible (la veracidad de un principio científico debe poder

ser replicado en otras condiciones).

Para generar el método científico se tomaron en cuenta enfoques

opuestos para el desarrollo de nuevo conocimiento: el método inductivo

y el método deductivo. El primero permite extraer características gene-

rales a partir de la observación de sucesos particulares, los cuales deben

repetirse en varias condiciones y una gran cantidad de casos. Y el segundo

se comienza con la argumentación general y se examina las posibilidades

para llegar a una conclusión específica. De manera general el método in-

ductivo se encuentra enfocado a la creación de nuevas teorías, el método

deductivo está enfocado a comprobarlas.

Las etapas principales del método son:

I.
Observación: 
Observar algún fenómeno natural específico.

II.  
Pregunta: 
Las observaciones llevan a plantearse una pregunta  

específica.

III.  
Hipótesis: 
Mediante el conocimiento previo u otras investigaciones

relacionadas a la pregunta, se plantea una hipótesis o predicción que

responde a la pregunta planteada y explica el fenómeno natural.

IV.  
Experimento: 
Realizar una serie de experimentos donde se controla

las variables para poder tener resultados válidos.

V.  
Conclusión: 
Ya recolectados los datos que se generaron en el experi-

mento, se analizan los resultados mediante técnicas estadísticas. Estos

resultados permiten validar o refutar la hipótesis.

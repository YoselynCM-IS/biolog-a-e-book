replican de manera independiente a la célula y se clasifican según su color en:

 

a )  
Sin color: aminoplastos, oleoplastos y proteinoplastos.

 

b )  
Otros colores: cromoplastos.

 

c )  
Colores tenues: etioplastos y proplastidios.

 

d )  
Colores verdes: cloroplastos.
